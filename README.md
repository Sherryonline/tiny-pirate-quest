# Tiny Pirate Quest

Tiny Pirate Quest is a simple one-player browser adventure game. The player controls a young pirate, collects coins, avoids the crab enemy, and opens the treasure chest to win.

## Game Concept

The game is inspired by cheerful pirate adventure themes and simple island exploration. It is designed to be easy to play, lightweight, and suitable for vibe coding practice.

## Features

- One-player 2D browser game
- Pirate character movement
- Coin collection
- Crab enemy obstacle
- Treasure chest win condition
- Health and score display
- Simple HTML, CSS, and JavaScript structure

## How to Play

1. Open `index.html` in a browser.
2. Move the pirate using arrow keys or WASD.
3. Collect all coins on the island.
4. Avoid the crab enemy.
5. Open the treasure chest after collecting enough coins.
6. Win the game when the chest is opened.

## Testing Checklist

- Player can move correctly.
- Player cannot move outside the game area.
- Coins are collected only once.
- Score updates after collecting coins.
- Enemy movement works correctly.
- Health decreases when touching the enemy.
- Game Over appears when health reaches 0.
- Chest cannot be opened before collecting enough coins.
- You Win appears after opening the chest.
